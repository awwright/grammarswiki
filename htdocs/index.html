About the Standard Grammar Catalog

Grammars.wiki is a comprehensive resource designed to make formal grammars and technical standards accessible to developers, engineers, and anyone working with file formats and protocols. We break down the complex rules that define how data is structured and shared across systems, helping you understand how the web and modern software really work behind the scenes.

What you'll find

  • File Formats: Understand the structure of key formats like CSV, PDF, MP4, SVG, and more. Our entries guide you through how files are encoded and parsed, making it easier to write or troubleshoot file processing code.
  • Protocols: Dive into the details of Internet communication protocols such as TCP/IP, HTTP, and WebSockets. Our step-by-step breakdowns explain how data is transmitted, ensuring you can build or debug networked applications confidently.
  • Grammars & Syntax: Learn about the formal grammars that define everything from programming languages to markup languages. We provide real-world examples so you can apply these rules to your own work, whether you're writing code that parses a document or sending a network request.
  • Interactive Examples: Try out interactive examples and explore live code snippets to see these formats and protocols in action. You'll get hands-on experience without having to piece together scattered information from multiple sources.
